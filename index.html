Utilisation de Prism.js - Code highlighter

  • Télécharger le CSS et le JS.
  • Importer les fichiers CSS et JS dans le fichier HTML (plusieurs choix possibles: langages web principaux, sélection de quelques langages ou tous).
    Voir Download Prism.js pour les langages, plugins et versions dev ou prod.
  • Déclarer la classe langage.
  • Remplacer le < par son entity (voir dans le code HTML).
  • Ci-dessous, quelques exemples de rendu dans plusieurs langages.

HTML

<!DOCTYPE html>
<html>
  <head>
    <link href="themes/prism.css" rel="stylesheet" />
  </head>
  <body>
    <section class="language-html"></section>
      <pre>
        <code class="highlight"><h1>hello world !</h1></code>       
      </pre>
    </section>
    ...
    ...
    ...
    <script src="prism.js"></script>
  </body>
</html>

CSS
